# Personalized Content Feed App

A modern personalized content feed web application built with Next.js, Redux, and Tailwind CSS.  
The app fetches and displays user-preferred content dynamically from multiple APIs and allows search and content organization.

# 📌 Features

- Personalized content based on user category preference
- Fetches content from multiple APIs
- Search functionality
- Drag-and-drop content reordering
- Favorites section
- Responsive modern UI
- Loading and error states
- Redux state management

---

# 🧱 Tech Stack

- Next.js 14 (App Router)
- React
- Redux Toolkit
- Tailwind CSS
- TypeScript
